FORGOT PASSWORD:
A "Forgot password?" link on the login page now sends a 6-digit OTP code
to derravi6@gmail.com, which you use to set a new password at
/admin/reset-password. For this email to actually be delivered, set these
environment variables wherever you deploy the app (otherwise the code is
just printed to the server console/logs instead of emailed):

    MAIL_SERVER=smtp.gmail.com
    MAIL_PORT=587
    MAIL_USERNAME=your_sending_gmail@gmail.com
    MAIL_PASSWORD=an_app_password_not_your_real_password
    MAIL_SENDER=your_sending_gmail@gmail.com

(Use a Gmail "App Password", not your normal Gmail password — Google
requires this for SMTP. Generate one at https://myaccount.google.com/apppasswords)
